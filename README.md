# Customer ETL Pipeline

## 📌 Objective

Build an ETL pipeline to clean and process customer data.

## 🔄 ETL Process

### Extract

* Raw data ingested into `customers_raw`

### Transform

* Handled NULL values
* Standardized missing fields

### Load

* Clean data loaded into `customers_clean`

## 🛠️ Tools Used

* SQL Server

## 📂 Structure

* raw-data/ → Source data
* transform/ → Cleaning logic
* final/ → Final processed data

## 🚀 Learnings

* ETL workflow design
* Data cleaning techniques
* Writing transformation logic
